## MusicTube 

<br>

<br>

Simple Electron + Node.js YouTube Desktop Software.

<br>

<br>

## Getting Started

<br>

<br>

```bash

<br>

git clone https://github.com/VioletBlend/MusicTube
cd MusicTube
npm init -y
npm install
npm start

<br>

```

<br>

<br>

📦 Requirements

```

<br>

Node.js (v18 or later recommended)

npm

Git

<br>

```

<br>

<br>

<br>


🖥️ Run The Apps

<br>

<br>

After running npm start, the Electron window should launch automatically.
